# User App (Angular)

## 📌 Description
This is an Angular-based User Management Application that performs CRUD operations. It allows users to add, update, delete, and view user details.

## 🚀 Features
- Add new users
- Edit user details
- Delete users
- View user list
- Form validation

## 🛠️ Technologies Used
- Angular
- TypeScript
- HTML
- CSS

## ▶️ How to Run
1. Install dependencies:
   npm install

2. Run the project:
   ng serve

3. Open browser:
   http://localhost:4200

## 📁 Project Structure
- src/app/models
- src/app/services
- src/app/user-list
- src/app/users
